Wego Clone VinaJet - Project Structure
=====================================

Core Components:
- src/components/PriceTrendBar.tsx - Enhanced price trend bar with Switch toggle for tax/fee display
- src/components/FlightResults.tsx - Flight search results with integrated PriceTrendBar and Price Comparison Modal
- src/components/SearchForm.tsx - Flight search form with enhanced calendar UI
- src/components/ui/calendar.tsx - Enhanced calendar component with gradients and animations
- src/components/ui/switch.tsx - Switch component for toggle controls

API Integration:
- src/lib/vinajet-api.ts - VinaJet API client with token caching and progressive search

Main Pages:
- src/app/search-results/page.tsx - Search results page with round-trip support and date selection
- src/app/page.tsx - Homepage with flight search form
